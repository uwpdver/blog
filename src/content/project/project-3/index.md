## 💻 命令

所有命令需在项目根目录的终端中运行：

根据你的包管理工具选择替换 npm，例如 `npm`、`pnpm`、`yarn`、`bun` 等。

| 命令                      | 操作                                           |
| :------------------------ | :--------------------------------------------- |
| `npm install`             | 安装依赖                                       |
| `npm run storybook`       | 在 `localhost:6006` 启动本地开发服务器         |
| `npm run build-storybook` | 构建生产站点到 `./storybook-static/`                       |
